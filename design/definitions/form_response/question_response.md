# Definition

A response to a question

# Parameters
- question_response_id - The id of the question response
- form_response_id - The id of the form response the question answer belongs to
- collection_id - The collection id of the question definition
- question_symbol - The question symbol
- question_version - The version of the question
- response_value_text - The text value of the response
- response_value_number - The number value of the response
- response_value_boolean - The boolean value of the response
