# Definition

A response to a form.

# Parameters

- form_response_id
- form_response_group_id -- The id of the form response group. Used to allow multiple forms to be filled at once.
- collection_id -- The id of the collection the form is defined in.
- form_symbol  -- The symbol of the form definition
- form_version -- The version of the form
- user_id -- Id of the user that filled or was filled on-behalf of
- filling_user_id -- Id of the user that actually filled the form.
- status -- Status of the form  -- Draft or Submitted.
- started_timestamp -- Timestamp when the form filling was started
- submitted_timestamp -- Timestamp when the form was submitted.
