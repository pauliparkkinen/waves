# Definition

A communication from one user to other users.

# Properties

- Sender -- User id of the sender
- Recipients -- User ids or organisation ids of the recipients
- Status of the message -- draft | sent | deleted
- Time of creation
- Time of last edit
- Time of sending
- Previous communication -- Can be empty if first message
- Purpose -- A predefined purpose code or id
- Contexts -- For example the ids of questionnaires etc
- Free text content
- Attached documents
- Version number -- Enables editing of the messages while retaining context
